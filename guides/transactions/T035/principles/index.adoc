[[principles]]
= Principles and prerequisites

This chapter describes the principles and assumptions that underlie the use of Peppol Pre-award Catalogue request. The B2G (Business to Government) process 
is based on P002 - Procurement Document Access and the B2B (Business to Business) process on what is common between companies..

This BIS will identify, explain and justify the business requirements for the Pre-award Catalogue request. It
provides syntax bindings to OASIS UBL 2.5. It also includes a syntax implementation guide.
