[[principles]]
= Principles and prerequisites

This chapter describes the principles and assumptions that underlie the use of Peppol Pre-award Catalogue. The B2G process is
based on P003 - Tender Submission and the B2B process on what is common between companies.

This BIS will identify, explain and justify the business requirements for the Pre-award Catalogue. It
provides syntax bindings to OASIS UBL 2.5. It also includes a syntax implementation guide.
