[[transportation-status-type-code]]
= Transportation status codes

The Transportation status type code identifies the type of status that is requested. 
The following codes can be used:

* 3=Deviations
* 4=All execution statues

The Transport Execution Status Code is used in the Transportation Status to identify the actual status on the shipment, consignment or transport handling unit that is being reported.
