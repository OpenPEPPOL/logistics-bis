[[principles]]
= Principles and prerequisites

This chapter describes the principles and assumptions that underlie the use of PEPPOL Pre-Award Catalogue. The B2G (Business to Government) processt is
based on P003 - Tender Submission and the B2B (Busness to Business) process on what is common between companies.

This BIS will identify, explain and justify the business requirements for the Pre-award Catalogue. It
provides syntax bindings to OASIS UBL 2.5. It also includes a syntax implementation guide.
