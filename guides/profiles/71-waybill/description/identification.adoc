[[identification]]
= Document identification

The Waybill is identified by the Document ID on header level. The Document ID is also referred to as the Booking identifier.
In addition there are several other ID's that can be included in the Waybill. These are used as references from this Waybill to ID's created by the different parties.

* CarrierAssignedID
* ConsignorAssignedID
* ConsigneeAssignedID
